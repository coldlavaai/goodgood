# How to Import Blog Posts to Shopify

## Quick Instructions

1. Go to Shopify Admin → **Online Store** → **Blog posts**
2. Click **"Add blog post"** for each article below
3. Copy the **Title**, **Content**, and **Meta description** from each section
4. Set **Author** to your name
5. Add relevant **tags** (suggested for each post)
6. Click **Save**

## After Adding All Posts

### 1. Set up your blog navigation
- Go to **Navigation** in Shopify admin
- Add a link to your blog in the main menu
- Link text: "Blog" or "Skin Health Blog"
- Link: Your blog URL (usually /blogs/news or /blogs/the-good-good-blog)

### 2. Enable blog comments (optional)
- Go to **Settings** → **Apps and sales channels**
- Enable Shopify comments or install Disqus

### 3. Set featured images
- Use product photos, lifestyle images, or relevant stock photos
- Recommended size: 1200x630px (works well for social sharing)

### 4. Internal linking
After all posts are live, go back and add internal links between related posts:
- Link "CBD for Skin" from "Eczema Treatment" post
- Link "TSW Guide" from "Eczema Treatment" post
- Cross-link related content

### 5. Add "Related Posts" section
Consider installing a Shopify app like "Related Blog Posts" to show similar articles at the bottom of each post.

## SEO Tips

1. **Use the exact URL handles provided** - they're optimized for search
2. **Don't change the titles** - they include target keywords
3. **Keep meta descriptions under 160 characters** - all provided descriptions are optimized
4. **Add alt text to images** - describe what's in the image + include keyword
5. **Publish consistently** - Add one post every few days to show activity
